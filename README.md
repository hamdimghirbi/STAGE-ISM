# CRA_Expenses_AI_Connector

Serveur MCP piloté par IA (Claude) permettant aux consultants d'automatiser
la déclaration mensuelle de leurs notes de frais et jours travaillés.
L'assistant extrait les données des justificatifs, les classifie, et les soumet
au portail Portalite — avec validation humaine obligatoire avant toute soumission.
Stack : Python 3.11+, FastMCP, API Anthropic, Playwright.
